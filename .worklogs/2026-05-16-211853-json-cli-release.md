# Summary

Released `slopless@0.2.0` with a zero-config CLI that always emits JSON.

Users can now install only `slopless` and run `npx slopless` without installing `textlint` separately or writing `.textlintrc.json`.

# Decisions Made

- Added `src/cli.ts` as a thin wrapper around textlint's CLI.
- Injected `--preset slopless`, `--rules-base-directory`, and `--format json` inside the wrapper.
- Rejected user-provided `--format` / `-f` because output is intentionally JSON-only.
- Moved `textlint`, `@textlint/types`, and `@textlint/ast-node-types` into runtime dependencies so the package is self-contained for users.
- Added package `main` because textlint resolves preset package directories through `require.resolve`.
- Kept the wrapper thin; it does not parse, normalize, or reformat textlint findings.
